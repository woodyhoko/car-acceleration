# Car Acceleration Times — 0 to 60 mph (2000–2024)

An animated **Chart.js visualization** comparing 0–60 mph acceleration times across nine iconic car models over two decades. Tracks the rise of Tesla alongside traditional performance and economy cars.

**[▶ Live Demo](https://woodyhoko.github.io/car-acceleration)**

---

## What's Tracked

| Car | Category |
|---|---|
| Tesla Model S | EV / Luxury |
| Porsche 911 | Sports |
| BMW M3 | Sports |
| Chevrolet Corvette | Sports |
| Ford Mustang GT | Muscle |
| Subaru WRX STI | Performance |
| Mazda MX-5 | Roadster |
| Honda Civic Si | Hot Hatch |
| Toyota Camry | Family Sedan |

Data spans **2000–2024**, showing the dramatic acceleration improvement of EVs vs. gradual ICE improvements over the same period.

---

## Stack

- [Chart.js 3.7](https://www.chartjs.org/) — animated line chart
- Vanilla JS + HTML — zero build tools, single file
